Welcome to the administration system.

Control Panel

This is an unsecured application using maven, spring mvc, hibernate, thymeleaf, logback.

It has been adapted to include the todo theme for the administration area.

It was also amended to use thymeleaf in the view layer (instead of jstl) with a special adaptation of thymeleaf layouts.
